- Installieren Sie zunächst alle Abhängigkeit mit "install_dependencies.bat"
- Um die Bibliothek webcam-head-tracker, die Demo (webcam-head-tracker-demo.cpp) und Aufgabe_2 (Aufgabe_2.cpp) zu kompilieren, genügt ein Doppelklick auf "6_compile_all.bat"
- Die Executables finden Sie anschließend unter ./build/Release
- Zum Beenden von Aufgabe_2 ALT + F4 drücken
Hinweise:
- Für die Installation wird angenommen, dass Sie ebenfalls Übung 1 gemacht haben. Falls nicht, müssen zuerst die Pakete von Übung 1 installiert werden.
- Nach Kompilieren über "6_compile_all.bat" dürfen die Ordner/Dateien in "startpaket" nicht mehr verschoben werden.
